# Python简易记账本
小组协作：1组长 + 3组员
## 分工说明
1. 组员B：record.py 新增账单、本地JSON存储
2. 组员C：statistics.py 账单查询、月度收支统计
3. 组员D：main.py 主菜单、删除账单、程序入口
4. 组长：仓库搭建、代码审核、冲突合并、整体联调、文档维护

## 运行环境
Python 3.8 及以上，无需额外第三方库

## 启动方式
打开终端执行：
python main.py

## 功能列表
1. 录入收入/支出，自定义分类、日期、备注
2. 查看全部历史账单
3. 按月自动计算总收入、总支出、结余、分类开销
4. 删除任意一条账单
## 数据存储
账单自动保存至 bill.json 文件，关闭程序不丢失数据

## 协作规范
1. 禁止直接在main分支开发
2. 每个人在自己专属分支写代码，完成提交PR
3. 提交备注统一格式：完成XX功能 / 修复XXbug
